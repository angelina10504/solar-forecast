# Solar Generation Forecasting

End-to-end ML pipeline for forecasting solar power generation at Indian plant locations, with DSM penalty simulation tied to forecast accuracy.

## Status
Week 1 — AWS setup and project scaffolding.

## Stack
- Storage: AWS S3
- Querying: AWS Glue + Athena
- Modeling: SageMaker Studio (LightGBM, Prophet)
- Inference: AWS Lambda + API Gateway
- Frontend: Next.js + Vercel
- Region: ap-south-1 (Mumbai)
